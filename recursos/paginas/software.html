Empieza a programar en linea con el Editor Web de Arduino, guarda tus proyectos en la nube y mantente siempre actualizado a la ultima versión!

El IDE de Software Libre de arduino te lo pone mas fácil para escribir código y subirlo a la placa, funciona tanto en Windows como Mac OS X y Linux.

Accede a la colección de proyectos que la gente desarrolla en arduino, si crees que has conseguido realizar algo interesante, no dudes en compartirlo!
